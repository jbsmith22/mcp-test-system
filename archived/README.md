# Archived Files

This folder contains legacy files, older versions, and scripts that are no longer actively used but kept for reference.

## Contents

### Legacy Ingestion Scripts
- `ingest_nejm_100.py` - Original batch ingestion script
- `ingest_next_100.py` - Sequential ingestion script
- `ingest_new_nejm_articles.py` - Early ingestion implementation
- `manual_ingestion.py` - Manual article ingestion
- `nejm_text_ingest.py` - Text-based ingestion

### Deprecated Features
- `connect_real_ai_logic.py` - Early AI integration attempt
- `reingest_to_opensearch.py` - Legacy re-ingestion script

## Note

These files are kept for historical reference and may contain useful code patterns or configurations. However, they are not part of the current active system and may not work with the latest infrastructure.

For current functionality, use the files in:
- `aws-deployment/` for AWS cloud features
- `on-prem/` for local deployment features
- `testing/` for validation and testing
